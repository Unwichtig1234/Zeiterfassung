WICHTIG FUER GITHUB

1. Diese ZIP NICHT als ZIP-Datei in GitHub hochladen.
2. Diese ZIP zuerst auf deinem Computer entpacken.
3. Dann den INHALT dieses Ordners in dein GitHub-Repository hochladen.
4. Im Repository muss package.json direkt oben sichtbar sein.
5. Danach in GitHub auf Actions > Build Desktop Installer > Run workflow.
